Cougar Framework Pear Channel

Registering the channel: 
pear channel-discover alfmel.github.com/pear
Listing available packages: 
pear remote-list -c cougar
Installing a package: 
pear install cougar/package_name
Installing a specific version/stability: 
pear install cougar/package_name-1.0.0
pear install cougar/package_name-beta
Receiving updates via a feed: 
http://alfmel.github.com/pear/feed.xml
